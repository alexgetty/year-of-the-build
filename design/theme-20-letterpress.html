Building in public means every decision is permanent, every commit a statement. Like letterpress, there is no undo — only the next impression.

Year of the Build is a declaration: twelve projects shipped in twelve months, documented weekly. It is accountability made visible, craft practiced in the open. Each devlog entry is a woodblock — carved with care, pressed with intention.

The body text uses Source Serif Pro for readability across long-form content. Headlines are set in Playfair Display Black, channeling the bold authority of wood type specimens.

Small text for captions, metadata, and fine print. Quiet but legible.

All caps for labels and navigation
