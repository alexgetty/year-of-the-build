Year Of The Build

A mathematician's notebook. Precise but warm. Systematic but personal.

Concept

CODEX synthesizes four themes: the golden ratio precision of Ratio, the warm notebook feeling of Field Notes, the geometric confidence of Construct, and the architectural type stacking of Stack. The result is a design system that feels like opening a well-loved notebook filled with careful observations.

  • Golden ratio type scale
  • Fibonacci spacing (8, 13, 21, 34, 55, 89)
  • Single typeface: Space Grotesk
  • Warm paper + warm black
  • Dot grid background
  • Geometric accents, sparingly

Callouts
Note

The golden ratio appears throughout nature - from nautilus shells to galaxy spirals. We use it not as dogma, but as a starting point for visual harmony.

Caution

The accent color should be used sparingly. Its power comes from restraint. Use for links, highlights, and key geometric accents only.

Confirmed

The Fibonacci spacing scale has been validated across all components. Mathematical harmony is maintained.

Design Principles

  • Single typeface mastery
  • Constraint as freedom
  • Geometry with purpose

Build Process

  1. Establish the grid
  2. Define type scale (phi)
  3. Choose color sparingly
  4. Test on dot grid
  5. Ship with confidence

Pre-Launch

  • Type scale defined
  • Spacing system tested
  • Components built
  • Documentation complete
  • Launch announcement
