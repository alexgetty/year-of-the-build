const createGrid = (baseUnit: number = 8) => ({
  unit: baseUnit,
  spacing: (multiplier: number) => baseUnit * multiplier,
  columns: 12,
  gutter: baseUnit * 2,
  maxWidth: baseUnit * 150, // 1200px
});

const theme = {
  colors: {
    bg: '#FFFFFF',
    fg: '#000000',
    blue: '#0000FF',

  },
  transitions: {
    instant: '0ms',

  }

07
Callouts
Information
The grid system uses an 8px base unit. All spacing, sizing, and layout decisions must be multiples of this unit. The grid is law.
Warning
Breaking the grid is a capital offense. Every element must align. No exceptions. Mathematical harmony is non-negotiable.
Error
Critical failure. This callout is for breaking changes, deployment failures, and things that require immediate attention.
Success
Build complete. Deploy successful. Ship it. Move on to the next one.
